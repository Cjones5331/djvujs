Расширение для браузера DjVu.js Viewer

Имеется расширение для браузера DjVu.js Viewer, которое по сути является копией программы-просмотрщика, представленной на данном сайте. Расширение позволяет открывать ссылки на документы djvu прямо в браузере, не скачивая файл явно, и, конечно же, есть возможность просматривать файлы с локального диска.

На данный момент расширение доступно для браузеров

Mozilla Firefox

и

Google Chrome
